Evaluación heurística

Evaluación heurística

Párrafo

Estas se utilizan para evaluar una interfaz por inspección, donde un experto en usabilidad analiza la calidad del sistema utilizando los principios de Jakob Nielsen. El experto en usabilidad debe evaluar dando una puntuación según la escala de Likert a cada uno de los elementos que se toman en cuenta en los principios de Jakob Nielsen, a partir de los resultados que se obtengan se llega a las conclusiones de los posibles problemas que pueda tener la página web.

Pruebas de usabilidad con usuarios

Pruebas de usabilidad con usuarios

Párrafo

Permiten utilizar la observación y el análisis para evaluar la interfaz de una página web haciendo que los usuarios realicen tareas específicas mientras que son observados por investigadores que trabajan en la página web. El objetivo de esta es evaluar la usabilidad de un sitio web, donde se capten reacciones, confusiones, oportunidades para mejorar y con esto decidir si el producto está listo para ser lanzado. La mayor parte del tiempo los investigadores sueles omitir errores por el intenso contacto con el diseño de la página web, por eso son importantes las pruebas de usabilidad con usuarios, ya que permite tener distintas perspectivas donde se revelan problemas, aparecen oportunidades de mejora y se ve cómo sería el comportamiento del usuario en la página web.

Estudios de campo y observación de usuarios

Estudios de campo y observación de usuarios

Párrafo

La observación de campo tiene como objetivo entender el como los usuarios de la página web interactúan con esta y realizan tareas específicas y más específicamente conocen la función y usabilidad de la página web. Para estas se deben escoger una variedad de usuarios los cueles estarán en un sitio de observación en donde se les dará un tiempo para cumplir tareas específicas. La primera parte que se realiza es la observación donde se analiza como el usuario se maneja en la página web, como lo hace, que objetos utiliza, donde se ubica, etc. Posteriormente se realiza una entrevista acerca de lo que realizaron en la página web para completar la información recabada.
